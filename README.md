# Задание

Разработайте Spring Boot приложение с REST API для управления задачами (TODO-лист) и настройте полный CI/CD пайплайн с
автоматическим деплоем на виртуальную машину.

TODO-лист (список задач) — это приложение для управления задачами, где можно:

Создавать новые задачи (например: “Купить молоко”)
Просматривать список всех задач
Отмечать задачи как выполненные
Удалять или редактировать существующие задачи

## Требования к приложению:

REST API для CRUD операций с задачами:
o POST /tasks — добавить задачу
o GET /tasks — получить все задачи
o PUT /tasks/{id} — обновить задачу
o DELETE /tasks/{id} — удалить задачу
Хранение задач в памяти (без БД)
Валидация входных данных
Покрытие тестами ≥ 70%

### Советы по реализации

Для REST API используйте:
@RestController для обработки запросов
List<Task> для хранения задач в памяти
@Valid для валидации входных данных
В тестах проверяйте:
Все CRUD операции
Коды ответов HTTP
Корректность возвращаемых данных
Для Docker:
Используйте .dockerignore
Оптимизируйте размер образа
Добавьте HEALTHCHECK
Для деплоя:
Настройте проброс портов на роутере
Проверьте доступность SSH
Реализуйте стратегию blue-green
Успешного выполнения задания! Помните, что главное — понять принципы работы CI/CD!
